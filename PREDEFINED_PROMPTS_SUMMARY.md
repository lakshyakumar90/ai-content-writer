# Image Studio - Predefined Prompts Feature

## Overview
Added predefined prompts for different content types to help users quickly generate images suited for various social media platforms and use cases.

## Categories Implemented

### 1. LinkedIn Post (5 prompts)
- Professional business meetings
- Success and growth themes
- Team collaboration
- Professional workspace aesthetics
- Business partnerships

### 2. Instagram Post (5 prompts)
- Aesthetic flat lays
- Travel and lifestyle photography
- Minimalist aesthetics
- Fashion and lifestyle content
- Food photography

### 3. YouTube Thumbnail (5 prompts)
- Bold attention-grabbing designs
- Before/after comparisons
- Question/curiosity-driven compositions
- Tech and gaming aesthetics
- Wealth and success themes

### 4. Blog Header (5 prompts)
- Abstract geometric designs
- Lifestyle and cozy aesthetics
- Food blog imagery
- Productivity themes
- Travel inspiration

### 5. Social Media Story (5 prompts)
- Vertical format quote designs
- Behind-the-scenes moments
- Product showcases
- Daily routine aesthetics
- Interactive content designs

## UI Changes

### Content Type Selector
- Dropdown menu to select content type
- "Custom Prompt" option for freeform entry
- Categories update dynamically

### Suggested Prompts Section
- Scrollable list of 5 curated prompts per category
- Click any prompt to auto-fill the prompt field
- Easy to scan and select
- Hover effects for better UX

### Improved Prompt Input
- Changed from single-line input to textarea
- Accommodates longer, more detailed prompts
- Minimum height of 80px
- Better for editing and reading

## How It Works

1. **Select Content Type**: Choose from 5 predefined categories
2. **Browse Prompts**: Scroll through 5 suggested prompts for that category
3. **Click to Use**: Click any prompt to automatically fill the prompt field
4. **Customize**: Edit the prompt or use as-is
5. **Generate**: Click generate to create your image

## Benefits

- **Time-Saving**: Pre-written professional prompts
- **Quality**: Prompts optimized for each content type
- **Learning**: Users can learn prompt engineering by example
- **Flexibility**: Can still write custom prompts
- **Consistency**: Ensures brand-appropriate imagery
